## 🎮 Gameplay

### Comment créer un nouveau monde ?
1. Appuyer sur le bouton **+** en bas à droite
2. Entrer un nom pour votre monde
3. Choisir le mode de jeu (Infini ou Compétitif)
4. Appuyer sur **Créer**

### Combien de mondes puis-je créer ?
Maximum **10 mondes** par compte.

### Comment supprimer un monde ?
1. Sur l'écran des mondes, appuyer sur l'icône **🗑️** du monde
2. Confirmer la suppression

---

## ☁️ Sauvegarde Cloud

### Mes mondes sont-ils sauvegardés automatiquement ?
**Oui**, si vous êtes connecté avec votre compte Google :
- Sauvegarde automatique après chaque action
- Synchronisation cloud automatique
- Badge **"À jour"** (vert) quand la sync est OK

### Que signifient les badges colorés ?
- 🟢 **"À jour"** : Monde synchronisé avec le cloud
- 🟠 **"À synchroniser"** : Synchronisation en attente
- 🔴 **"Erreur cloud"** : Échec de synchronisation
- ⚪ **"Local uniquement"** : Pas de cloud sync (non connecté)

### Que faire si j'ai une erreur cloud ?
1. Vérifier votre connexion internet
2. Appuyer sur l'icône **🔄** pour réessayer
3. Si l'erreur persiste, se déconnecter et se reconnecter

### Puis-je jouer sans connexion ?
**Oui**, mais :
- Les mondes restent en local uniquement
- Pas de synchronisation cloud
- Risque de perte si vous désinstallez l'app

## 🔄 Multi-appareils

### Puis-je jouer sur plusieurs appareils ?
**Oui**, si vous utilisez le même compte Google :
1. Se connecter sur le 2ème appareil
2. Les mondes apparaissent automatiquement
3. Appuyer sur **Télécharger** si besoin

### Comment récupérer mes mondes après réinstallation ?
1. Réinstaller l'application
2. Se connecter avec le même compte Google
3. Les mondes réapparaissent automatiquement

## 🎯 Stats et Progression

### Que signifient les stats affichées ?
- **Trombones** : Nombre de trombones en stock
- **Vendus** : Nombre total de trombones vendus
- **Argent** : Argent disponible

### Comment voir mes backups ?
Le nombre de backups est affiché sous la date de modification.
Pour restaurer un backup, appuyer sur l'icône **🔄**.

## 🔐 Compte et Connexion

### Dois-je créer un compte ?
**Non**, vous pouvez jouer sans compte, mais :
- Pas de sauvegarde cloud
- Pas de multi-appareils
- Risque de perte des données

### Comment me connecter ?
1. Appuyer sur l'icône de profil en haut à droite
2. Choisir **Se connecter avec Google**
3. Sélectionner votre compte Google

### Comment me déconnecter ?
1. Appuyer sur l'icône de profil
2. Choisir **Se déconnecter**
3. Confirmer

## 🐛 Problèmes Courants

### L'app plante au démarrage
1. Fermer complètement l'app
2. Vider le cache (Paramètres > Apps > PaperClip2 > Vider le cache)
3. Redémarrer l'app

### Mes mondes ont disparu
1. Vérifier que vous êtes connecté avec le bon compte
2. Appuyer sur **Actualiser** (icône 🔄 en haut)
3. Vérifier votre connexion internet

### La synchronisation ne fonctionne pas
1. Vérifier votre connexion internet
2. Se déconnecter et se reconnecter
3. Appuyer sur **Retry** (icône 🔄) sur le monde
